# dotfiles

Dotfiles &amp; scripts used on macOS

## Initial setup of new Mac

- run `./scripts/setup.sh`
- put secrets in `~/secrets/`
- run `./scripts/setup-secrets.sh`
- install
  - XCode (from AppStore)

### iTerm2:

- import `iterm.drey.json` profile

### VSCode:

Plugins:

- prettier
- Tailwind CSS IntelliSense
- eslint
- css modules
- code-spell-checker
- GitLens
